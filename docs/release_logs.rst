Release Logs
============

Version 0.1.0 (alpha)
---------------------

Initial alpha release of **simpleLOMs**.

- Core LOM fitting interface via ``fit_lom`` with Optimized, Foster, and Analytical models.
- CPW resonator networks, hanger resonators, and hybrid series + hanger designs.
- Modular subsystem fitting for multi-resonator circuits.
- Design helpers for targeting frequency and loaded quality factor.
- Tutorials, schematics export, and Sphinx documentation site.
